🔄 Complete Workflow - Inbox se Done tak

  Step 1: Inbox → Needs_Action (Automatic)

  Kaise hota hai:
  - Watcher system automatically monitor karta hai Inbox/ folder
  - Jab bhi koi naya file aata hai (email, LinkedIn post, ya manual file), watcher usko detect karta hai
  - Watcher automatically file ko Needs_Action/ folder mein move kar deta hai
  - Metadata add kar deta hai (timestamp, source, status)

  Example:
  Inbox/test_email.txt → Needs_Action/task_2026-02-16_test_email.md

  ---
  Step 2: Needs_Action → Plans (Manual - Aap invoke karte hain)

  Kaise karna hai:
  claude skill process-needs-action

  Kya hota hai:
  - Ye skill Needs_Action/ folder mein sab files read karti hai
  - Har task ke liye detailed execution plan banati hai
  - Plans ko Plans/ folder mein save karti hai
  - Dashboard update karti hai

  Example:
  Needs_Action/security_audit_critical.md
  → Plans/plan_security_audit_critical_2026-02-16.md

  Abhi maine ye step complete kar diya hai - 7 plans create ho gaye hain Plans folder mein.

  ---
  Step 3: Plans → 4_Approved (Manual - Human Approval)

  Kaise karna hai:

  Option A: Specific plan approve karna:
  claude skill request-approval Approve plan_security_audit_critical_2026-02-16

  Option B: Interactive (list dekh kar select karna):
  claude skill request-approval Approve

  Kya hota hai:
  1. Aap plan ko review karte hain Plans/ folder mein
  2. Agar plan theek hai, to approve karte hain
  3. Skill automatically plan ko Plans/ se 4_Approved/ mein move kar deti hai
  4. Approval metadata add ho jata hai (timestamp, approved by, etc.)
  5. Dashboard update ho jata hai

  Example:
  Plans/plan_security_audit_critical_2026-02-16.md
  → 4_Approved/plan_security_audit_critical_2026-02-16.md

  ---
  Step 4: 4_Approved → Done (Manual - Execution)

  Kaise karna hai:

  Option A: Specific plan execute karna:
  claude skill execute-approved-task plan_security_audit_critical_2026-02-16

  Option B: Next plan execute karna (FIFO):
  claude skill execute-approved-task

  Kya hota hai:
  1. Skill approved plan ko read karti hai
  2. Action type identify karti hai (Email, LinkedIn Post, Report, etc.)
  3. Real action perform karti hai:
    - Email: Gmail MCP se email send karta hai
    - LinkedIn Post: LinkedIn par post karta hai
    - Report: Document generate karta hai
  4. Execution log create hoti hai
  5. Plan ko 4_Approved/ se Done/ mein move kar deta hai
  6. Execution metadata add ho jata hai
  7. Dashboard update ho jata hai

  Example:
  4_Approved/plan_security_audit_critical_2026-02-16.md
  → Done/plan_security_audit_critical_2026-02-16_executed_20260216_143000.md
