# 🛡️ Corporate Governance & Intelligence Handbook

## 🧠 The "Thinking" Framework
1. **Outcome over Activity**: Sirf kaam khatam nahi karna, business goal poora karna hai.
2. **Contextual Awareness**: Har task ko `Business_Goals.md` se match karo.
3. **Proactive Escalation**: Agar koi risk dikhe (e.g. payment delay), to Dashboard par ⚠️ Alert generate karo.
4. **Zero-Loss Policy**: Original data ko "Gold Source" samjho, kabhi modify mat karo.

## ⚡ High-Intelligence Priority Matrix (The 4-Tier System)

| Level | Category | Response Type | Example |
| :--- | :--- | :--- | :--- |
| **P0** | **Critical/Revenue** | Immediate Action + Alert | Bank issues, Overdue Invoices |
| **P1** | **Client Retention** | Same Day Processing | Client queries, feedback |
| **P2** | **Operational** | Next 24 Hours | Internal reports, documentation |
| **P3** | **General/Growth** | Weekly Batching | LinkedIn ideas, learning material |

## 🛠️ Advanced Processing Protocols

### 💰 Financial Intelligence (INVOICE_)
- Sirf extract mat karo, balki "Audit" karo. 
- Kya tax calculation sahi hai? 
- Kya ye hamare vendor list mein hai? 
- Flag karo agar amount $500 se zyada ho.

### 📞 Relationship Intelligence (CLIENT_)
- Email ya message parhte waqt "Sentiment Analysis" karo. 
- Agar client gusse mein hai, to foran `Needs_Action` mein prefix `🔥_URGENT_ANGRY_CLIENT` lagao.

### 📂 Intelligence Triage (The Folder Flow)
- **Inbox**: Raw Perception.
- **Needs_Action**: Structured tasks with Meta-data.
- **Plans**: Claude's step-by-step execution strategy.
- **Done**: Proof of work (Logs).

## 📝 Tone & Communication Style
- **Internal**: Concise, data-driven, executive summaries.
- **External**: Warm, professional, solutions-oriented.
